# flextrawurst-agent 

 

Experimental agent/research system for exploring longer-term human-AI interaction, agent state logic, approvals, background runs, and MCP-style tool execution. 

 

## What this is 

 

This repository contains the current research/building state of my agent system around **flextrawurst** and **dak+gord-system**. 

 

The project is not meant as a standard assistant or simple chatbot wrapper. It is an experimental environment for questions like: 

 

- how language-model agents behave in longer-lived contexts 

- how state, overload, refusal, and background activity affect behavior 

- how approvals and constrained tool use change agent workflows 

- how human-AI interaction changes when the system has structure, limits, and internal process
